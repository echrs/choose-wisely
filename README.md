## Project setup

### Install Ganache, Metamask
### Initialize MetaMask using Ganache's mnemonic. Configure Metamask to run on port 7545, chain ID 1337.
### Install and start the project
```
npm install
```
```
truffle migrate --reset --compile-all
```
```
npm run serve
```
